Space Invaders

-- Javadoc
To see the Javadoc use 'gradle javadoc' on terminal, Documentation will be located in 'build/docs/javadoc/index.html.'
index.html is the root of the documentation of the code.

Note: When running 'gradle javadoc' on terminal warnings will appear this is because not all functions are commented.
Comments on Javadoc will pertain to design patterns in assignments.

Available Javadoc comments:
GameEngine
GameWindow
MementoPackage
ObserverPackage
PrototypePackage
SingletonPackage
Player.copy()
Bunker.copy()
Enemy.copy()
